## 高优先级 ASR 校正规则（用户人工确认，优先级高于模型猜测）

以下为用户人工确认的高优先级校正规则。模型在诊断（pre_correction）与校正（corrected）时，**应优先采用本节规则，而不是自行猜测新人物/新词**。每条规则都附带触发语境，须结合上下文判断后再校正。

- **徐绍良昵称**：正确写法为「阿良」或「良仔」。近音误听 **阿楠、阿亮、阿梁、阿凉、阿娘、娘** 若上下文指向徐绍良，应校正为「阿良」；用户已确认剧中没有「阿楠」这个人物，不要据此臆造新角色。
- **康炳仁昵称**：正确写法为「阿仁」或「仁哥」。近音误听 **人哥** 若上下文指向康炳仁，应校正为「仁哥」；铺子名以「仁记…」开头。
- **李小霞昵称 / 阿霞**：用户人工确认 EP10 开头「阿霞」指鄭麗芳饰演的「李小霞」，不是赵雅芝饰演的「朱敏芝 / 阿芝」，也不是陳比莉饰演的「夏鳳」。出现「阿霞」且上下文是徐绍良女朋友、深夜回家、打电话给阿良等语境时，应保留为「阿霞」或关联「李小霞」，**不得自动改成「阿芝」「朱敏芝」或「夏鳳」**。
- **走片**（戏院术语）：指跑片员跑片 / 送递胶片拷贝。戏院、放映、影业语境中出现近音时按「走片」处理，不要理解成「跑路」。
- **做擺牌 / 擺牌**（戏院宣传杂务）：指摆放广告牌 / 宣传牌。近音误听 **走白牌、走 ba pai** 若在戏院杂务语境（与跑片、宣传、戏院广告并列），应重点判断为「做擺牌」，不要当成赌牌 / 牌照 / 走私之类。
- **大蓉 / 中蓉 / 細蓉**（面档云吞面规格）：近音误听 **大傭、大勇** 若在端面、叫面、面档上菜语境，应校正为「大蓉」，不要当成人名或「勇敢」之意；「中蓉」「细蓉」同理。
- **鹼水鮮 / 蝦餃鮮**（面档叫卖语境的人工确认词）：近音误听 **蛋最、蛋水、蛋嘴** 等若在面档叫卖语境，应重点考虑为「鹼水鮮」；近音误听 **蝦夠、蝦狗** 等应重点考虑为「蝦餃鮮」。不要把「蝦狗」「蛋嘴」当成真实食物名。仍无法确认时保留 uncertain 标记。

## 饮食 / 面档词

- 大蓉 / 中蓉 / 細蓉（简体：大蓉 / 中蓉 / 细蓉）：香港云吞面/面档叫法，指不同分量规格的云吞面。ASR 常把"大蓉"误听为"大傭""大勇"——若上下文是端面、叫面、面档上菜，应将"大傭""大勇"修正为"大蓉"，不要当成人名。"中蓉""细蓉"同理，按云吞面规格理解。简体译文根据上下文译为"大份/中份/小份云吞面"。（EP09 index 348 属此类）
- 鹼水鮮 / 蝦餃鮮（简体：碱水鲜 / 虾饺鲜）：云吞面、银丝面、面档叫卖语境中的面类/食物称呼。面档叫卖语境中，近音"蛋最""蛋水""蛋嘴"应重点考虑为"鹼水鮮"；近音"蝦夠""蝦狗"（及"蝦狗鮮"）应重点考虑为"蝦餃鮮"。不要把"蝦狗""蛋嘴"当成真实食物名。若原声更接近"鹼水鮮"，应译为"碱水鲜"；仍无法确认可保留 uncertain 标记。（EP09 index 294/329 出现相关疑点，需结合上下文判断）
