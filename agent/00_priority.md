# 资料优先级

本项目用于为《大亨》生成字幕。ASR 校正、DeepSeek 校正和翻译都必须遵守以下资料优先级。

## 优先级顺序

1. 前8集人工精校字幕
2. 官方角色名结构化资料 `agent/characters_official.json`
3. 项目人工确认的专有名词和 ASR 校正规则
4. Wiki 简介、故事大纲等公开资料
5. 网友剧情简介和观片线索
6. 模型根据当前上下文自行判断

## 冲突处理

- 前8集人工精校字幕、官方角色名、用户人工确认术语优先于公开资料、网友线索和模型猜测。
- Wiki、故事大纲和网友线索只用于辅助理解剧情，不可作为专有名词最终依据。
- 模型不得自行改写已确认的人名、地名、公司名、固定称谓和高优先级 ASR 校正规则。
- 未确认的专有名词应保守处理；无法判断时保留稳妥写法，并标记或记录为待人工复核。
