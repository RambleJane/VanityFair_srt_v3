# 参考字幕

`simplified_human/01.srt` 至 `08.srt` 是前 8 集人工精校简体字幕，仅作为离线参考资料，不直接参与当前集切分。

可从中提取角色称谓、常见译法、语言风格、旧版人工字幕表达习惯、主题曲切分参考，以及剧情和人名线索。程序生成的衍生画像建议写入 `cache/reference_profile/`，或在确认需要长期维护时写入 `agent/reference_profile_01_08.json`。

完整 SRT 可能涉及版权内容，默认只在本地保留，并由项目根目录 `.gitignore` 排除。仓库可保留本说明和 `simplified_human/.gitkeep`。

资料冲突时，按 `agent/00_priority.md` 的顺序处理；不得用低优先级线索覆盖人工精校字幕或官方角色资料。
